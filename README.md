### Projeto em Destaque: Sistema de Cadastro de Pets

Uma aplicação prática do que venho aprendendo no meu Hub de Estudos. Este sistema CLI (terminal) foi desenvolvido em **Java** e realiza CRUD completo de pets. O foco principal foi dominar a **lógica de manipulação de arquivos.txt** para persistência manual. A arquitetura foi estruturada seguindo os princípios de **DDD (Domain-Driven Design)**, organizando o código em camadas claras de `domain`, `service` e `repository` para facilitar a futura refatoração para PostgreSQL e Springboot.

**[Ver Código e Documentação no GitHub](https://github.com/Kaua-Henrique1/sistema-cadastro-java)**
